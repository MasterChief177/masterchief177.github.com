Creation of a Virtual Firewall on a NAS System (Proof of Concept)

Project Goal: Cost- and resource-efficient implementation of a Sophos XG firewall as a virtual machine on an existing NAS system to significantly increase network security.

1. The Challenge & The Goal: The primary challenge was to establish a fully functional enterprise-class network security solution without having to purchase additional physical hardware (such as a dedicated firewall router). The main goal was the development and successful implementation of a zero-cost solution using existing NAS hardware and virtualization to separate the internal (LAN) and external (WAN) network via a single physical network port.

2. My Role & Impact (Highlighting Diversity): In this project, I acted as a system integrator and network architect. My focus was on solving the central configuration challenge, namely the mapping of two network zones (WAN and LAN) to a single physical port of the NAS system.

Network Design and Virtualization: I implemented complex virtual bridging (for the internal LAN) and a NAT interface (for the WAN/Internet connection). This innovative network masking was crucial to meet the requirements of the virtual firewall.

System Integration and Configuration: I was responsible for the installation of the Sophos XG Firewall VM and the precise assignment of the virtual interfaces in the hypervisor and then in the firewall operating system.

Problem Solving and Validation: The project demonstrates my ability to overcome restrictive hardware environments (single-port NAS) through creative software and virtualization solutions, thus creating a fully functional security barrier.
